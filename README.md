# CloudServer

## 動機

隨著科技網路的迅速發展，人們使用雲端上傳檔案也越來越普及，導致雲端商業產品也越來越多，因此我們想要架設屬於自己的私有雲端伺服器後，就可以將檔案上傳至自己的雲端伺服器上。

## 硬體材料

| 名稱           | 數量 | 來源     |
| -------------- | ---- | -------- |
| Raspberry Pi 3 | 2    | MOLi     |
| 2TB 隨身碟     | 1    | 自備 |
